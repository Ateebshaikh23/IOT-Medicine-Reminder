# **App Name**: SmartMed

## Core Features:

- Smart Notifications: Receive medicine reminders via browser alerts based on set schedules.
- Medication History: Automatically log and display medication history, tracking doses and times using LLM tool to confirm dose timings .
- Flexible Schedules: Allow users to set daily, weekly, or custom medication schedules.
- User Dashboard: Display a dashboard showing upcoming doses, history, and adherence.
- Dark/Light Mode Toggle: Implement a toggle for dark and light mode to improve user experience.
- Live Demo: Offer a live demo of the dashboard.

## Style Guidelines:

- Primary color: Indigo/Violet (#6C63FF), symbolizing intelligence and healing.
- Background color: Light blue (#F0F4FF), a heavily desaturated version of the primary hue for a calm and trustworthy feel.
- Accent color: Turquoise (#4CD7D0), an analogous color with higher brightness and saturation to create visual interest and highlight important actions.
- Font: 'Inter', a grotesque-style sans-serif font that gives a modern and neutral look to the website; suitable for headlines or body text.
- Use Heroicons, Phosphor, or custom SVG icons for a consistent and modern look.
- Use Framer Motion or AOS.js for subtle animations to improve user engagement.
- Implement a clean and modern layout with clear sections for the hero, features, how it works, and testimonials.
